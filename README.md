# 🦜 手养鹦鹉之家

一个超酷炫的鹦鹉展示与管理网站！

## ✨ 功能特点

- 🎨 **酷炫视觉效果** - 流动星光背景 + 霓虹配色
- 🦜 **鹦鹉卡片展示** - 图片/视频、价格、手养天数一目了然
- 📊 **实时统计** - 鹦鹉总数、平均天数、总价值
- 🎬 **视频切换** - 点击播放按钮查看鹦鹉视频
- 📝 **管理后台** - 轻松上传鹦鹉信息
- 💾 **本地存储** - 数据保存在浏览器 localStorage

## 🚀 快速启动

```bash
# 方法 1: 使用启动脚本
./start.sh

# 方法 2: 直接使用 Python
python3 -m http.server 8080

# 方法 3: 直接用浏览器打开 index.html
```

然后在浏览器访问：`http://localhost:8080`

## 📖 使用说明

### 添加鹦鹉

1. 点击右上角「管理鹦鹉」按钮
2. 填写鹦鹉信息：
   - 名字
   - 价格（元）
   - 手养天数
   - 上传照片（必填）
   - 上传视频（可选）
   - 描述（可选）
3. 点击「保存鹦鹉信息」

### 查看鹦鹉

- 首页展示所有鹦鹉卡片
- 鼠标悬停卡片有炫酷动画
- 点击视频按钮切换图片/视频
- 悬停时显示删除按钮

### 删除鹦鹉

- 鼠标悬停到鹦鹉卡片
- 点击右下角垃圾桶图标
- 确认后删除

## 💡 提示

- 数据保存在浏览器本地存储，清除缓存会丢失数据
- 图片和视频会转换为 base64 存储，建议文件大小不要太大
- 每 30 秒会有一只小鹦鹉飞过屏幕 🦜

## 🛠️ 技术栈

- HTML5 + CSS3 + JavaScript
- Canvas 动态背景
- localStorage 数据存储
- Font Awesome 图标
